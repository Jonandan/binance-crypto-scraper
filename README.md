
Overview
This project extracts real-time cryptocurrency market data from the Binance API and stores it in a structured CSV format.

Features
- Fetches live crypto market data
- Extracts symbol, price, and trading volume
- Saves data into a CSV file
- Simple and efficient API-based scraping

Technologies Used
- Python
- requests
- CSV handling

Data Extracted
- Symbol (e.g., BTCUSDT)
- Last Price
- Trading Volume
